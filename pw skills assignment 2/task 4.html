HTML tags:-

  1. Heading tags(h1,h2,h3,h4,h5,h6)--->Used to define the title or sub title of a webpage.

  2. paragraph tag--->Used to write a paragraph in a webpage

  3. hr tag--->display a horizontal line.

  4. br tag--->used for a line break.

  5. div tag--->defines a division or section in the html page.

  6. dl tag--->defines the description list.

  7. form tag--->Used to create a form.

  8. table tag--->used to create a table.

  9. img tag--->used to embed an image.

  10. style tag--->used to define CSS for a document.
